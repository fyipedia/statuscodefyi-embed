Status Codes

Quick Start

One script tag. Status Codes embedded in 30 seconds.

1

Add the script

Include the embed script from jsDelivr CDN anywhere in your HTML.

2

Place the element

Add a div with the data-statuscodefyi attribute.

3

Done

The widget renders automatically — fully isolated by Shadow DOM with auto dark mode.

HTML 
<!-- Step 1: Add embed script -->
<script src="https://cdn.jsdelivr.net/npm/statuscodefyi-embed@1/dist/embed.min.js"></script>

<!-- Step 2: Place the widget -->
<div data-statuscodefyi="entity"
     data-slug="example-codes"
     data-style="technical"
     data-theme="auto"></div>

Widget Options

Attribute Values Default Description
data-statuscodefyi entity · glossary · search · compare · faq required Widget type to render
data-slug string required Entity slug from statuscodefyi.com
data-style technical · modern technical Visual style variant
data-theme light · dark · sepia · auto auto Color theme (auto follows OS preference)
data-lang en · es · ja … en Display language
data-no-snippet true false Disable rich snippet injection into host page
